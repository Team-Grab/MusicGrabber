# ◢◤ Music Grabber v1.0.0

**Music Grabber** es un orquestador de preservación digital diseñado para gestionar descargas de música con una arquitectura técnica resiliente y una interfaz TUI (Terminal User Interface) de estética cyberpunk.

## ✨ Características Principales

* **6 Modos de Orquestación:** Desde álbumes cronológicos hasta disolución de playlists y carpetas para DJs (Modo Mix).
* **Arquitectura de Resiliencia:** Sistema de persistencia atómica y protocolo de Rollback automático en caso de cierres inesperados.
* **Gestión Anti-Ban:** Tres perfiles de velocidad para evitar bloqueos 403.
* **Auto-Bootstrap:** El programa gestiona y actualiza automáticamente sus propias dependencias (`yt-dlp` y `ffmpeg`).
* **Interfaz TUI:** Experiencia de terminal inmersiva construida con el framework Textual.

## 🚀 Instalación y Uso

1. Descarga el instalador oficial `MusicGrabber_Setup_v1.0.0.exe` desde la sección de **Releases**.
2. Ejecuta el instalador y sigue las instrucciones.
3. Al iniciar, el programa verificará e instalará los motores necesarios en tu carpeta de usuario (`AppData/Local/MusicGrabber`).

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Interfaz:** Textual TUI
* **Motores:** yt-dlp & FFmpeg
* **Build:** PyInstaller & Inno Setup
